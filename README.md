# MysticBot — Эзотерический Telegram-бот

Бот для предсказаний, раскладов Таро, нумерологии, гороскопов и финансового календаря.

## Возможности
- **Карты Таро**: Виртуальные расклады, интерпретация
- **Хиромантия**: Анализ фотографий ладони (CV)
- **Нумерология**: Расчёты по дате рождения, имени
- **Гороскопы**: Ежедневные, недельные, месячные прогнозы
- **Финансовый календарь**: Астрологические рекомендации для финансов
- **Сонник**: Толкование снов
- **Руны**: Гадание на рунах

## Структура проекта
```
MysticBot/
├── bot/                 # Основной код бота
├── ml/                  # Машинное обучение (хиромантия, стилизация)
├── data/                # Общие данные (JSON колоды, изображения)
├── docker-compose.yml   # Docker Compose
├── Dockerfile          # Docker образ
├── requirements.txt    # Зависимости Python
├── .env.example       # Пример переменных окружения
├── alembic.ini        # Настройки миграций
└── README.md
```

## Быстрый запуск
```bash
# 1. Клонировать репозиторий
git clone <repo>
cd MysticBot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить переменные окружения
cp .env.example .env
# отредактировать .env, добавить токен бота

# 4. Запустить бота
python bot/main.py
```

## Docker
```bash
docker-compose up -d --build
```

## Конфигурация
- `BOT_TOKEN`: Токен Telegram-бота
- `DATABASE_URL`: PostgreSQL подключение
- `REDIS_URL`: Redis для кэширования и FSM
- `OPENAI_API_KEY`: Для LLM интерпретаций (опционально)
- `PERPLEXITY_API_KEY`: Для расширенных предсказаний
